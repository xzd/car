2、公路运费查询接口

接口调用请求说明

HTTP请求方式：get
HTTP请求地址：/index/freight/type/1

请求参数说明

参数 数据类型 是否必须 说明
start number Y 出发地（区编号), 默认满洲里（150781）
end number Y 目的地（市编号）, 比如杭州（330100）
正确返回示例
{
    code: 1,
    data: {
        price: "50-80" // 价格区间
    },
    msg: "操作成功"
}
错误返回示例
{
    "code": 0,
    "data": "",
    "msg": "参数错误"
}

7、发货方端 - 货源发布接口

接口调用请求说明

HTTP请求方式：post
HTTP请求地址：/goods/add

请求参数说明

参数 数据类型 是否必须 说明
start_province number Y 出发地 - 省 code
start_city number Y 出发地 - 市 code
start_district number Y 出发地 - 区县 code
start_province_name string Y 出发地 - 省
start_city_name string Y 出发地 - 市
start_district_name string Y 出发地 - 区县
end_province number Y 目的地 - 省 code
end_city number Y 目的地 - 市 code
end_district string Y 目的地 - 区县 code
end_province_name string Y 目的地 - 省
end_city_name string Y 目的地 - 市
end_district_name string Y 目的地 - 区县
car_length string Y 车长（多选用英文逗号隔开）: 4.5米,6.0米
car_type string Y 车型: 高栏
goods_type string Y 货物类型: 木材
goods_num number Y 货物数量
goods_unit string Y 货物单位: 吨
money number N 运费金额: 元/单位
repeat number Y 20分钟重发15次, 默认选中: 1 选中, 0不选
public number Y 是否公开, 默认公开: 1公开, 0不公开
正确返回示例
{
    "code": 1,
    "data": "",
    "msg": ""
}
错误返回示例
{
    "code": 0,
    "data": "",
    "msg": "发布失败，请重试"
}
